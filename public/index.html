Data sources & methodology

Store locations on this map were collected using public-facing APIs exposed by each platform. The figures shown are a lower bound based on what those APIs returned at scrape time.

Actual counts can differ for many reasons (availability, geography, API changes, timing). What you see here is at least what each platform’s own APIs surfaced—not an exhaustive census.

  • Zepto — Data scraped 14–15 March.
  • Blinkit — Data scraped 15–17 March.
  • Swiggy (Instamart) — Data scraped 18–19 March.

Blinkit: store coordinates were trilaterated from API data. Most markers are within about 0–50 metres of the actual dark-store location; anything beyond that is shown as an imprecise marker.
